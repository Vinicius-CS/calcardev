# Calcardev
Sistema de análise de crédito desenvolvido para teste na empresa Calcard.

# Construção
 **Requisitos**
 - Eclipse Oxygen 3
 - JDK Java
 - RS2XML
 
 **Construindo**
 
1. Clone ou faça o download do repositório:

 `git clone https://github.com/VinyciusC/Ayffir`

2. Abra o projeto no Eclipse Oxygen 3.
